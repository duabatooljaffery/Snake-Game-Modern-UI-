**NOVA SNAKE – MODERN SNAKE GAME**

**PROJECT DESCRIPTION**
NOVA SNAKE is a modern implementation of the classic Snake game built using HTML, CSS, and JavaScript. The game features smooth gameplay, responsive controls, dynamic level progression, and high-score storage using LocalStorage. This project demonstrates strong front-end development and game logic skills.

---

**FEATURES**
• Classic Snake gameplay
• Real-time keyboard and touch controls
• Score and level progression system
• Increasing speed with levels
• Collision detection (wall and self)
• Pause and restart functionality
• Responsive design for mobile and desktop
• Best score saved using LocalStorage

---

**TECHNOLOGIES USED**
• HTML5 – Structure of the game
• CSS3 – Styling and animations
• JavaScript (Vanilla JS) – Game logic
• HTML5 Canvas API – Rendering graphics
• LocalStorage – Saving high scores

---

**SKILLS DEMONSTRATED**
• JavaScript Game Development
• HTML5 Canvas Rendering
• Event Handling
• Collision Detection
• Game Loop Implementation
• Responsive Web Design
• UI/UX Design
• Local Storage Integration
